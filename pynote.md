### 字符串 str
```
    isinstance() 函数: 判断字符串类型;
    eg.
        x = 'abc'
        y = 123
        # 返回 True
        instance(x, str)
        # 返回 False
        instance(y, str)
```

> 小写 str.lower()
    str.lower()

> 大写 str.upper()
